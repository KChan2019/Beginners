HTML Cheatsheet

Definiton:

HTML is a markup language. It describes the structure of web pages.

Always remember to do these things when starting:

  • Start the first lines with "<!DOCTYPE html>" followed by "<html>" to have a valid HTML document
  • Make sure to indent code to make it clearer to read
  • Don't forget to close the tags </>

Different HTML tags and definitions:

<body>: Wraps around all the content that is displayed on the page

<title>: Controls what shows on browser tag

<head>: Container for all the head elements

<h(number)>: Headings - numbers indicate size

<img scr>: Adds image and locates image

<ol> (followed by <li>): Numbered list

<ul> (followed by <li>: Bullet point list

<a href>: Links to URLs

<p>: Paragraphs

<i>: Italitise text

<strong>: Bold text

<input>: To get information from users

&lt &gt: to show the < and > symbols

&amp: to show the & symbol
